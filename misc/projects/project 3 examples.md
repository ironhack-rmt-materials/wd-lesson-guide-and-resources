
# Project 3 examples


Eventure: https://eventure-eventdiscoverapp.netlify.app/
- (Geethu)
- Models: User + Event + Booking


Mealionaire: https://mealionaire.netlify.app/
- (Sevda)
- Models: User + Meal + Order
- Great UX, responsive
- Bonus: searchbar, filter by categories, pagination...


Memory Lane: https://memory-lane-web.netlify.app/
- (Kateryna)
- Great functionality + responsive
- Functionality: create capsule, share capsule, see public capsules, comment...
- Models: User + Capsule + Comment + Invitation


EventHive: https://eventhive-flame.vercel.app/
- (Carlos)
- Models: User + Event + Comment


Roomie: https://roomie-home.vercel.app
- (Anderson)
- An app to keep track of shared flat tasks and expenses
- Functionality: create a flat, invite other people as members of a flat, create tasks, create expenses...
- Models: User + Flat + Expense + Task + Invitation
- Bonus: image upload, emails, charts


Tribbu: https://tribbu.vercel.app
- (Pau)
- An app to keep track of family/child tasks
- Functionality: create a tribbu (ie. a family), invite other people to a tribbu, create tasks...
- Models: User + Tribbu + Event + Invitation
- Bonus: social login, emails, calendar... 
- Note: the app is in Spanish
